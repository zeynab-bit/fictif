# 🏡 ProjetImmo

  Description

*KSA immobiliere* est un site web de présentation et de gestion de biens immobiliers. 
Il permet aux utilisateurs de consulter différentes offres de logements 
(appartements, villas, studios, duplex, etc.), d'obtenir des informations détaillées sur chaque bien et de contacter 
facilement l'agence immobilière.

Le projet a été développé en utilisant les technologies *HTML*, *CSS* et *JavaScript*, dans le but de proposer une 
interface moderne, intuitive et responsive.

---

## Fonctionnalités implémentées

*  Affichage de plusieurs biens immobiliers.
*  Présentation détaillée de chaque logement.
*  Galerie d'images des biens.
*  Affichage du prix, de la localisation et des caractéristiques.
*  Navigation entre les différentes pages du site.
*  Formulaire de contact avec validation des champs en JavaScript.
*  Interface responsive adaptée aux ordinateurs, tablettes et smartphones.
*  Menu de navigation simple et ergonomique.
* Mise en page moderne avec CSS.

---

## Technologies utilisées

* HTML5
* CSS3
* JavaScript
